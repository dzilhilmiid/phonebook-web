# Phonebook Web

Aplikasi atau Web phonebook sederhana menggunakan Vue 3 dan Firebase Firestore.

## Fitur

- Create Contact
- Read Contact
- Update Contact
- Delete Contact
- Search Contact
- Sort Contact (A-Z, Z-A)

## Tech Stack

- Vue 3
- Vite
- Firebase Firestore

## Cara Menjalankan

npm install
npm run dev
